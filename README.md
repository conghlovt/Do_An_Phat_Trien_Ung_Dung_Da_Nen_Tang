# Hệ thống quản lý khách sạn/homestay

## Công nghệ sử dụng

- Frontend: React Native + Expo
- Backend: Node.js + Express
- Cơ sở dữ liệu: PostgreSQL
- Công cụ thiết kế: Figma, Draw.io, Visual Paradigm

## Cấu trúc dự án
- Back-End: API Server & Database Schema.
- Front-End: Ứng dụng di động và giao diện Web.
